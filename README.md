# Westmarch

Specs voor het personal-handbook-ecosysteem naar het Conduction-model: één
handbook als hub die documentatie uit alle persoonlijke repos (open source +
homelab) at build time aggregeert. Dit repo bevat de openspec-changes
(`personal-handbook-openspec/`) — inventarisatie, docs-contract en
handbook-portal — plus de northstar en roadmap in `openspec/project.md`.
Uitvoering verloopt voorstel-eerst via `personal-handbook-openspec/PROMPT.md`.

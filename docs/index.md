# Handbook

Eén hub die de documentatie van alle persoonlijke repos at build time
aggregeert. De docs leven in de project-repos zelf (`/docs`, volgens het
[docs-contract](conventies.md)); dit handbook bevat géén projectkennis,
alleen de kaart.

## Projecten (publiek)

- **Wanderer** — sovereignty checker voor organisaties (NLnet-traject)
- **estafette** — PoC-overdraagbaarheids-assessment (NLnet-traject)
- **wordsworth** — sovereign documentpijplijn; eerste habitat-afnemer

zeef (Woo-convergentietool, NLnet), habitat (agent-platform) en homelab
